Версия сайта БЕЗ оценок
========================

Что изменено:
- сайт читает только коллекцию Firestore `openings`;
- коллекция `ratings` больше не читается;
- вкладка `Профиль`, никнейм, кнопки оценок и общий рейтинг удалены;
- импорт загружает только данные об опенингах/эндингах;
- стартовые данные: 2094 OP/ED, 0 оценок.

Какие файлы заменить в GitHub:
1) index.html
2) oped-app.js
3) firebase-import.html
4) firebase_import_helper.js
5) op_ed_initial_data.js
6) firestore.rules — его нужно не просто загрузить в GitHub, а ещё скопировать в Firebase Console -> Firestore Database -> Rules -> Publish.

ВАЖНО:
Файл firebase-config.js НЕ трогай и НЕ заменяй, если ты уже вставил туда Firebase config.

Если импорт ещё НЕ запускался:
1) Замени файлы выше.
2) В Firebase Console открой Firestore Database -> Rules.
3) Вставь текст из firestore.rules и нажми Publish.
4) Открой https://ТВОЙ_НИК.github.io/op-ed-orden/firebase-import.html
5) Нажми "Начать импорт опенингов".
6) После успеха удали firebase-import.html из GitHub.

Если старый импорт уже запускался:
1) Просто замени index.html и oped-app.js.
2) Сайт перестанет читать оценки, поэтому станет быстрее.
3) Повторно импорт лучше не запускать, чтобы не получить дубли, если старый импорт создавал документы другим способом.

Если в Firestore уже есть коллекция ratings:
- можно оставить её как есть: новая версия сайта её не читает, поэтому она не тормозит сайт;
- если хочешь удалить физически, это отдельная операция в Firebase Console.
